# Documentation Website

This website is built using [Docusaurus](https://docusaurus.io/), a modern static website generator.

## Installation

```bash
npm install
```

## Local Development

```bash
npm start
```

This command starts a local development server and opens up a browser window. Most changes are reflected live without having to restart the server.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory and can be served using any static contents hosting service.

## From Root Directory

You can also run these commands from the root of the monorepo:

```bash
# Development
npm run docs:dev

# Build
npm run docs:build

# Preview production build
npm run docs:preview
```

## Features

- **Interactive Wallet Scanner**: The homepage features a fully functional Solana wallet privacy scanner
- **Comprehensive Documentation**: All guides, API references, and CLI documentation
- **Custom Styling**: Matching the original VitePress design with Source Sans 3 font family
- **SEO Optimized**: Full meta tags, Open Graph, and Twitter Card support
