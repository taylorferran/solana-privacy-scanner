# Placeholder Images Notice

The following images need to be created and placed in this directory:

## Required Images

1. **og-image.png** (1200 x 630 px)
   - Open Graph image for social media previews
   - Used by Facebook, LinkedIn, WhatsApp, etc.

2. **twitter-image.png** (1200 x 630 px)
   - Twitter Card image for Twitter/X previews
   - Can be same as OG image or optimized for Twitter

3. **apple-touch-icon.png** (180 x 180 px)
   - iOS home screen icon
   - Also used on other Apple devices

4. **favicon-32x32.png** (32 x 32 px)
   - Standard favicon for browsers
   - Shows in browser tabs

5. **favicon-16x16.png** (16 x 16 px)
   - Smaller favicon for high-DPI displays
   - Shows in browser UI elements
